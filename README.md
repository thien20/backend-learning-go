# Learning Stages

- `01-understand-request-flow`: learn the layering and one read path.
- `02-build-crud-api`: extend the same layout into a more realistic CRUD API shell.
- `03-measure-before-optimizing`: measure first, then improve obvious bottlenecks.

Each folder is a standalone mini-repo with its own `go.mod` so a student can work through the stages one at a time.
